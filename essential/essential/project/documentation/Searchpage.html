Search page


The search page of BookPlanet has links to the home page, the signup and login pages and the documentation of the application. Entering the title of any book will give the results below; the first 10 results from Google Books API are returned. A call is made to the API depending on the title of the book and the data returned is in JSON format. The values returned are put into different varaibles and presented in HTML.
