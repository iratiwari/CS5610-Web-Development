Introduction


BookPlanet is a small social platform and search site for book lovers. Users can search for books they like and get detailed information on the books. They can also review books and add to favourites, this is done by logging in to the website. Users can also get information about the author of the book.

The following pages have been created for BookPlanet :

Homepage : The main homepage of the application. This contains the link to the search page , the login and signup pages.

Search page : In the search page, the user can enter the name of any book. The search is done as per the title of the book and the results are displayed.

Book details page : If the user is on the search page and clicks on the image or the title of the book, they will be taken to the book details page. Here the book image, ISBN and a summary of the book are shown.

Author details page : If the user is on the search page and clicks on the link for the author of the book, they will be taken to the author details page. Here the image of the author and other details of the author are shown.

Author books page : If the user is on the author details page and clicks on the link "Books by this author" , they will be redirected to the author books page. This displays all the books by a particular author. Only the images are shown for those books.

Author books details page : Clicking on any of the images on the author books page will take the user to the details page for that book. This page is different from the book details page since the data is being extracted from a different API. The title , image and summary of the book is shown.

Signup page : Clicking on the signup link on the top right of any of the pages will take the user to the signup page, where they can register for the website. On this page the user is made to enter their basic details which will be stored in the database.

Login page : Clicking on the login link on the top right of any of the pages or trying to review a book ; or add it to the favourites will take the user to the login page. Here they can login using their e-mail address and password. The details will be matched with those stored in the database and the user will be redirected to the profile page.

Profile page : Once the user is logged in ; they are taken to the profile page. This page will display the details of the user; and the books they have added to their favourites. Users can also see the reviews they have made for various books.
