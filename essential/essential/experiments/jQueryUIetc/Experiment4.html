Experimenting with animations using CSS keyframes

Animation using Keyframes

About the Experiment

In this experiment, I learned using css animation utility keyframes.

Code Snippet

CSS 
        #animation
        {
            text-align:center;
            width:400px;
            height:150px;
            background:#000000;
            color:#ffffff;
            position:relative;
            font-weight:bold;
            font-size:40px;
            padding:10px;
            animation:animation 5s infinite;
            -moz-animation:animation 5s infinite;
            -webkit-animation:animation 5s infinite;
            border-radius:5px;
            -webkit-border-radius:5px;
        }
          
        @keyframes animation
        {
            0%   {left:0px;}
            25%  {left:200px; background-color: green;}
            50%  {left:400px; background-color: red;}
            75%  {left:600px; background-color: yellow;}
            100% {left:0px; background-color: pink;}
        }
          
        @-webkit-keyframes animation
        {
            0%   {left:0px;}
            25%  {left:200px; background-color: green; margin-top: 50px;}
            50%  {left:400px; background-color: yellow;margin-top: 0px;}
            75%  {left:600px; background-color: red;margin-top: 50px;}
            100% {left:0px; background-color: pink;}
        }
          
        @-moz-keyframes animation
        {
            0%   {left:0px;}
            25%  {left:200px; background-color: green;}
            50%  {left:400px; background-color: red;}
            75%  {left:600px; background-color: yellow;}
            100% {left:0px; background-color: pink;}
        }
