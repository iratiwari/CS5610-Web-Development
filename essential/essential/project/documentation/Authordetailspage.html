Author details page


The author details page has links to the home page, the signup and login pages and the documentation of the application. Clicking on the name of the author on the search page will bring the user to the author details page. This data is extracted from the Goodreads API.The data is in XML format. The name of the author, image, date of birth and a short biography are shown. Clicking on "books by this author" will take the user to the author books page.
