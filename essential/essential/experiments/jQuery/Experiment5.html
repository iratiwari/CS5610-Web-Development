Experimenting with jQuery draggable and resizeable

1. Draggable

The image can be dragged and dropped anywhere on this screen.


Experiment Details

The dragging is handled by the function .draggable().

Code Snippet

            $(function () {

            $("#draggable").draggable();

            });

2.Resizable

We can resize the image by dragging from the boundaries.


Experiment Details

The resizable event is handled by resizable() funcion

Code Snippet

            $(function () {

            $("#resize").resizable();

            });

About the Experiment

In this experiment, I implemented jQuery interactions draggable and resizeable for 2 different images, by making functions for draggable and resizeable elements.
