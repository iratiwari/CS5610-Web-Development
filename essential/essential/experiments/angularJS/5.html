About the experiment

This experiment was performed to learn to implement tables using AngularJS
AngularJS invokes customersController with a $scope and $http object. $scope is the application object (the owner of application variables and functions). $http is an XMLHttpRequest object for requesting external data. $http.get() reads static JSON data from http://www.w3schools.com/website/Customers_JSON.php. If success, the controller creates a property (names) in the scope, with JSON data from the server. This data was then displayed after applying styling to it. I used | orderBy : 'Country' to order the names by their Country names.

Code Snippet

Javascript:
mynoteapp.js : 
             var app = angular.module("myNoteApp", []);
mynotecontrol.js : 
            app.controller("myNoteCtrl", function ($scope) {
            $scope.message = "";
            $scope.left = function () { return 500 - $scope.message.length; };
            $scope.clear = function () { $scope.message = ""; };
            $scope.save = function () { alert("Note Saved"); };
            });
HTML : 
            <body ng-app="myNoteApp" ng-controller="myNoteCtrl">
            <textarea ng-model="message" cols="50" rows="10"></textarea>
            <p> <button ng-click="save()">Save</button>
            <button ng-click="clear()">Clear</button>
            </p>
            <p>Number of characters left: <span ng-bind="left()"></span></p>
