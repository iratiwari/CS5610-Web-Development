Setup NodeJS server on OpenShift

INSTALLING NODEJS

  • Install nodejs from here
  • Add your nodejs home to PATH env variable
  • Create a new directory to put all your server code in
  • In the same directory install express using npm install express
  • List of other modules that can be installed using nmp are available here

OPENSHIFT SETUP for NodeJS

  • Sign up for an account in Openshift
  • Select Nodejs from available list
  • Complete the form and create a website by the given domain name and namespace
  • Follow the instructions given here to prepare your machine to be able to make changes to your application
  • Inside your server application directory, create a file called .gitignore, and add node_modules in it, inorder to not track it in GIT.

About the Experiment

In this experiment, I have created a basic page using server.js

  • var express = require('express');
    var app = express();

    require is used to include express into the nodejs server

  • app.use(express.static(__dirname + '/public')); is used to bind 'public' folder with all requests for static content.

  • app.get() is used to bind a response to a GET request with the matching URL.

  • Finally app.listen(port, ip) makes the server listen to incoming requests on given ip address and port number.

Code Snippet

server.js 
var express = require('express');
var app = express();
app.use(express.static(__dirname + '/public'));
app.get('/', function (req, res) {
res.send('Hello. Welcome to my OpenShift Page.');
});
var ip = process.env.OPENSHIFT_NODEJS_IP || '127.0.0.1';
var port = process.env.OPENSHIFT_NODEJS_PORT || 3000;
app.listen(port, ip);
