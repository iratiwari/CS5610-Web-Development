Angular JS Tables

Reading of static data from another web page, applying CSS to it and using orderby filer

{{ x.Name }} {{ x.Country }}

About the experiment

This experiment was performed to learn to implement tables using AngularJS
AngularJS invokes customersController with a $scope and $http object. $scope is the application object (the owner of application variables and functions). $http is an XMLHttpRequest object for requesting external data. $http.get() reads static JSON data from http://www.w3schools.com/website/Customers_JSON.php. If it succeeds, the controller creates a property (names) in the scope, with JSON data from the server. This data is then displayed after applying styling to it. I have used | orderBy : 'Country' to order the names by their country names.

Code Snippet

Javascript : 
        <script>
        function customersController($scope, $http) {
        $http.get("http://www.w3schools.com/website/Customers_JSON.php")
        .success(function (response) { $scope.names = response; });
        }
        </script>
HTML : 
        <div ng-app="" ng-controller="customersController">
        <table>
        <tr ng-repeat="x in names | orderBy : 'Country'">
        <td>{{ x.Name }}
        <td>{{ x.Country }}
        </tr>
        </table>
        </div>
CCS : 
        table, th, td {
            border: 1px solid grey;
            border-collapse: collapse;
            padding: 5px;
        }
        table tr:nth-child(odd) {
            background-color: #f1f1f1;
        }
        table tr:nth-child(even) {
            background-color: #ffffff;
        }
