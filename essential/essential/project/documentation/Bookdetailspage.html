Book details page


The book details page has links to the home page, the signup and login pages and the documentation of the application. Clicking on the book title or the image of the book on the search page will bring the user to the book details page. This data is extracted from the Google Books API. The title, book image, ISBN and book summary are shown. Clicking on add to favourites or add a review will take the user to the login page.
