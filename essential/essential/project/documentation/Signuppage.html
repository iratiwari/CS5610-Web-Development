Signup page


The Signup page has links to the home page, the search page, the login page and the documentation of the application. Clicking on signup on any page will take a user to this page. Also, cliking on "Sign Up" in the login page will bring the user to this page. The user is asked for their first and last name, e-mail and password to register for BookPlanet. These values are inserted in the database table users. The e-mail and password are later used for authenticating the user.
