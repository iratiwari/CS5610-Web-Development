Author books details page


The author books details page has links to the home page, the signup and login pages and the documentation of the application. Clicking on any of the book covers on the author books page will take the user to a page about details of the book. These details are extracted from Goodreads API in XML format; hence this page is slightly different from the book details page. The title, image and description of the book are shown here. Clicking on the add to favourites or add a review button will redirect the user to the login page of BookPlanet.
