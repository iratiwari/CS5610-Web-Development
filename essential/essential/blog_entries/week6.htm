Week 6 Blog

I learned AngularJS this week. I implemented the following experiments this week:

  • 1. Basics of AngularJS

    • This experiment was performed to learn AngularJS basics. AngularJS starts automatically when the web page has loaded. The ng-app directive tells AngularJS that the <div> element is the "owner" of an AngularJS application.


  • 2. Experimenting with filters in AngularJS

    • In this experiment I learned to use a filter in AngularJS.


  • 3. Experimenting with httprequest in AngularJS

    • In this experiment I learned to make an httprequest using AngularJS.


  • 4. Experimenting with tables in AngularJS

    • In this experiment I learned to implement tables using AngularJS. $http.get() reads static JSON data from http://www.w3schools.com/website/Customers_JSON.php. If it succeeds, the controller creates a property (names) in the scope, with JSON data from the server. This data is then displayed after applying styling to it. I have used | orderBy : 'Country' to order the names by their country names.


  • 5. Experimenting with an application of AngularJS

    • In This experiment I learned to make and save a module in AngularJS and create an alert for the same.
