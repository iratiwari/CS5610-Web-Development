About the Experiment

In this experiment, I have created a static HTML page and loaded it on the remote server at OpenShift

I used app.use(express.static(__dirname + '/public'));to bind 'public' folder with all requests for static content.

Code Snippet

server.js 
var express = require('express');
var app = express();
app.use(express.static(__dirname + '/public'));
var ip = process.env.OPENSHIFT_NODEJS_IP || '127.0.0.1';
var port = process.env.OPENSHIFT_NODEJS_PORT || 3000;
app.listen(port, ip);
