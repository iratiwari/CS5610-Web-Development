Login page


The Login page has links to the home page, the serach page, the Sign Up page and the documentation of the application. Clicking on login on any page will take a user to this page. Also, clicking on "Login" in the sign up page will bring the user to this page. Also if a user tries to add a book to their favourites or to add a review to any book, they will be redirected to the login page. The users are made to enter their e-mail and password in order to login. These values are matched to those already stored in the database and the user is logged in (or not) accordingly.
