About the Experiment

In this experiment, I learned about playing an audio or video file on a page in HTML5 using the tags for the same. Used bootstrap to make the experiment more presentable.

Code Snippet

HTML 
            <h1>Audio Video Implementations</h1>
            <hr />
            <h3>Playing an audio on a webpage</h3>
            <audio controls>
            <source src="../../source/aud.mp3" type="audio/mp3">
                </audio>
            <hr />
            <h3>Playing a video on a webpage</h3>
            <video width="400" height="300" controls>
            <source src="../../source/vid.mp4" type="video/mp4">
            </video>
